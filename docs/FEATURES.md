# Project Features: Mise-en-place

## 1. Recipe Store

- **Import/Entry:** Manual entry or import from external websites.
- **Account-based:** Recipes are stored under the user's account.
- **Multipliers:** Apply recipe multipliers (e.g., double or triple a recipe).
- **Nutrition:** Automatically calculate nutrition information.
- **On-the-fly Edits:** Swap ingredients to update nutrition/calorie tracking in real-time.

## 2. Meal Planning

- **Calendar View:** Plan recipes in advance.
- **Scheduling:** Add recipes to specific dates.

## 3. Pantry Monitoring

- **Inventory Management:** Add/remove items manually.
- **Auto-Decrement:** As recipes are marked "cooked" in the calendar, pantry items are automatically reduced.

## 4. Shopping List

- **Auto-Generation:** Items missing for planned meals are added to the shopping list.
- **Minimum Quantities:** Ensures enough is purchased for all planned meals.
- **Staples:** Flag items as "staples" that always appear on the shopping list.
