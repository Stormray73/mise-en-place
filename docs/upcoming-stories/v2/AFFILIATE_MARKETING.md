# Upcoming Stories: Affiliate Marketing

## Story: "Shop this Recipe" Integration

As a user, I want to quickly purchase ingredients for a recipe so that I can prepare for my meal plan.

### Acceptance Criteria

- [ ] AC 1: Intelligent matching of recipe ingredients to affiliate platform product IDs.
- [ ] AC 2: "Shop now" buttons generated with affiliate tracking tags.
- [ ] AC 3: UI to toggle affiliate links on/off in the Recipe View.

### Technical Notes

- Target platforms: Amazon Associates, Instacart, or similar grocery delivery APIs.
- Deferred to v2.0 to focus on core acquisition.
