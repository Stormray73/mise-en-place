# Upcoming Stories: UI/UX Refinement

## Story 10: Meal Planner Workflow

As a chef, I want a more flexible meal planning workflow so that I can adjust my schedule easily.

### Acceptance Criteria

- [ ] Users can delete a planned meal from the calendar.
- [ ] When creating a meal, users can immediately search for and add recipes in the same modal.
- [ ] Drag-and-drop support for moving meals between days (Bonus).

---

## Story 11: Dashboard & Landing Page

As a new user, I want to understand what Mise-en-place does so that I can decide to sign up.

### Acceptance Criteria

- [ ] Dashboard: Remove extraneous navigation buttons; focus on "Quick Actions" (Add Recipe, Plan Meal).
- [ ] Landing Page: Showcase a demo workflow, key features (Recipe Store, Planner, Pantry), and screenshots.
- [ ] Landing Page: Clear "Sign Up with Google" CTA.
- [ ] Pricing section (placeholder/coming soon).
