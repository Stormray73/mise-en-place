# Upcoming Stories: Dashboard Enhancements

## Story: Shopping List Quick Access

As a chef, I want to see my shopping list directly on my dashboard so that I can quickly check what I need or add items without navigating away.

### Acceptance Criteria

- [ ] Add a "Shopping List" widget or section to the main Dashboard Hub.
- [ ] Users can see a summary of the top items they need to buy.
- [ ] Users can add an item directly to the shopping list from this dashboard view.
- [ ] The "Add Item" interface should be streamlined for speed.

### Technical Notes

- Reuse logic from the main Shopping List page for adding items.
- Consider a "Quick Add" input field that defaults to common units or no unit.
