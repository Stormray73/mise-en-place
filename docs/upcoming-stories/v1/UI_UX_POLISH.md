# Upcoming Stories: UI/UX Polish

This epic tracks minor styling tweaks, layout improvements, and ergonomic refinements to ensure a high-quality user experience for v1.0.

## 1. Meal Planner UI Refinements

- [ ] **Widen UI:** Increase the width of the day columns in the Meal Planner to ensure there is enough space for recipe titles and the delete button.
- [ ] **Delete Button Visibility:** Ensure the delete button for meals is always easily accessible and visible without layout clipping.
- [ ] **Navigation Swap:** Switch the positions of the "Next Week" and "Current Week" buttons to follow more intuitive placement (e.g., Current Week on the left, Next Week on the right).
- [ ] **Meal Edit Button:** Add an "Edit" button to each planned meal that allows the user to quickly remove specific recipes from that meal slot.

## 2. Modal Ergonomics

- [ ] **Remove Redundant Cancel Buttons:** Remove the "Cancel" button that appears at the bottom center of modals if there is already a standard 'X' or if it is unnecessary for the workflow, reducing visual clutter.

## 3. General Layout

- [ ] **Dashboard Densification:** Continue to monitor and improve content density on the main dashboard hub for larger screens.
